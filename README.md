# Robofriends

RoboFriends is a simple web application built using Node.js, Express, and MySQL. It provides basic user registration and login functionalities using a MySQL database.

Features
User-friendly registration and login forms.
Secure password storage using MySQL.
Responsive design for seamless user experience across devices.
Installation
Clone this repository to your local machine.
Install Node.js and MySQL if you haven't already.
Run npm install to install project dependencies.
Create a MySQL database named robofriends_db.
Update the MySQL connection settings in index.js if necessary.
Run node index.js to start the server.
Access the application at http://localhost:3000 in your web browser.
